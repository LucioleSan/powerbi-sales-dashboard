# powerbi-sales-dashboard
J’ai conçu un rapport interactif Power BI permettant d’analyser les ventes d’une entreprise sous plusieurs angles : global, produits, géographique et temporel. L’objectif était de suivre les KPIs clés (chiffre d’affaires, marge, volume), d’identifier les meilleurs produits et régions, et de mettre en évidence les tendances de croissance.
